The front end of a website or application is the part users directly see and interact with. It involves designing and coding the user interface, ensuring elements
look appealing and function smoothly. Common front-end languages include HTML for structure, CSS for styling, and JavaScript for interactivity. Together,
they build responsive layouts, manage navigation, create animations, and handle user actions. Front-end developers aim to deliver a seamless user experience across various devices and browsers.


What is HTML?

HTML (Hypertext Markup Language) is the standard markup language used to structure the content of web pages. It
defines the page's layout, headings, paragraphs, links, images, and other elements. Browsers read HTML files and render the content visually for users,
making it the backbone of every website's structure.

What is CSS?

CSS (Cascading Style Sheets) is the language used to describe the presentation of a webpage's content. It controls the
visual aspects, including layout, colors, fonts, and spacing. By separating design from content, CSS makes it
easier to maintain and update a site's look without altering its HTML structure.

What is Javascript?

JavaScript (JS) is a programming language used to add interactivity and dynamic features to web pages. It allows
developers to modify content, handle user actions, create animations, and communicate with
servers, making websites more engaging and responsive.
